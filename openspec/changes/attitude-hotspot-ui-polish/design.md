# 设计说明

## UI Direction

本轮只做前台 UI polish，不改变数据接口。整体风格继续保持地图优先、克制、信息密度适中。

顶部简报使用横向数据带：`今日简报 | N 个热点 | 较昨日变化 | 主要话题`。变化值用中性色，不用红绿暗示好坏。

左侧结果列表使用态势色作为主色，话题只作为文字。卡片信息压缩为地区名、话题、趋势标签、态势值和热度条。

## Attitude Indicator

新增态势指针条，用于 hover 卡和右侧热点概览。位置计算：

`((weightedGoldstein + 10) / 20) * 100%`

空值落在 50%，显示“中性/混合”。颜色沿用现有态势色函数。

## Map Feedback

地图图例前台文案为“冲突倾向 / 中性混合 / 合作倾向”。底部说明保留为轻量风险提醒，不把 GDELT 作为主标题。

Marker 动效按趋势区分：

- 升温：较明显脉冲。
- 活跃：慢速弱脉冲。
- 冷却：无脉冲并略弱化。
- 暂无对比：静态。

使用 `prefers-reduced-motion` 禁用脉冲动画。

## Compatibility

不新增数据字段，不改变 `/api/hotspots` 返回结构。现有详情抽屉打开/关闭、地图拖拽、缩放、结果定位行为保持不变。
